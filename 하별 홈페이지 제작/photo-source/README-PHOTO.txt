하늘의별딸기 사진 넣는 폴더 안내

이 폴더에는 사진 6장만 넣으면 됩니다.

파일명은 아래처럼 맞춰 주세요.

1.jpg = 홈 첫 화면 대표 사진
2.jpg = 홈 소개 / 체험 사진
3.jpg = 예약 / 가격 대표 사진
4.jpg = 오시는 길 / 주차 안내 사진
5.jpg = 방문 가이드 대표 사진
6.jpg = SNS / 카카오톡 / 네이버 공유용 대표 사진

사진은 가능하면 가로형 JPG를 추천합니다.
JPG / JPEG / PNG가 가장 안전합니다.
HEIC, RAW(CR3)는 자동 변환이 안 될 수 있습니다.
너무 작은 사진은 피하고, 긴 변 1600px 이상이면 좋습니다.

사진을 넣은 뒤에는 프로젝트 루트에서 아래를 실행하면 됩니다.

./apply-photos.sh

실행이 어렵다면, 이 폴더에 사진만 넣고 Codex에게 "사진 반영해 줘"라고 요청해도 됩니다.
